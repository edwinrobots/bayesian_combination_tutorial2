Assembled 2019-04-11 by Brendan O'Connor (brenocon.com)

Data release for:

Snow, O'Connor, Jurafsky, Ng.  EMNLP 2018.
"Cheap and Fast -- But is it Good?  Evaluating Non-Expert Annotations for Natural Language Tasks"
https://www.aclweb.org/anthology/D08-1027

This builds on the dataset already released at:
 - http://ai.stanford.edu/~rion/annotations/
 - https://sites.google.com/site/nlpannotations/
Included within here as
 - all_collected_data.tgz

This directory contains some of the original texts for some of the tasks:
 - Affective text 
 - Recognizing Textual Entailment (rte)
 - Temporal ordering (tempeval)

It does not for:
 - Word similarity - the original data release has these (just word pairs)
 - WSD - the release has references to an external corpus
